# KIM-FLAGI design notes

- Treat CHMURNIK only as a visual reference. Never edit its files from this project.
- Keep the editorial visual language: Newsreader + Manrope, pale mineral surfaces, thin rules, restrained blue and terracotta accents, minimal rounding and shadow.
- Flag artwork must always preserve its intrinsic proportions. Presentation frames use centered `object-fit: contain`, independent `max-width` and `max-height`, and `overflow: hidden`; never size a flag from width alone.
- Write for a motivated reader, not for an assumed low reading level. Prefer technical vocabulary, comparisons, evidence and source links over simplified trivia.
- New learning features should connect back to monographs and preserve local-only progress: no accounts, advertising or tracking.
