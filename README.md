# FUN WITH FLAGS / KIM-FLAGI

Interaktywny, źródłowy atlas flag dla siedmioletniego odkrywcy. Zawiera 20 historii flag, filtrowany atlas, quiz i lokalnie zapisywane odznaki.

## Uruchomienie

```sh
npm install
npm run dev
```

## Kontrola jakości

```sh
npm test
npm run build
```

## Publikacja

Projekt jest przygotowany dla GitHub Pages pod adresem `/KIM-FLAGI/`.

Pliki flag pochodzą z Flagpedii/FlagCDN i bazują na Wikimedia Commons. Opisy mają odnośniki do źródeł rządowych i oficjalnych dokumentów bezpośrednio w aplikacji.
